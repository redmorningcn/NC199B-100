说明：
    NC199B-200为空检主板程序。
创建时间：
    2017-11-12
参考：
    《机车能耗通讯协议说明书》
主要任务：
    1、显示任务。         循环显示湿度、粉尘、VOC值，有故障则显示故障代码。任务为OSAL，周期500ms。
    2、数据存储任务。     循环存储采集的数据。数据结构长度128字节（stcFlshRecNCZ12），任务为OSAL，周期1min。
    3、测量模块通讯。     控制电磁阀打开，采集测量模块值，用modbus协议读取测量模块内容。测试周期10min，测试时间长度1min。
    4、无线模块通讯。     发送数据到地面服务器，可接受传输参数。如无数据15s发送一次，有数据立即发送。
    5、LED指示控制任务。  根据测量值，预设值指示。切换周期和测量模块通讯周期相同。
